Date

使用 Unity3D（C#）开发的第一人称探索解谜游戏项目。

负责关卡流程系统的设计与实现，统一管理关卡状态推进与解谜完成条件。

实现解谜房间的核心逻辑，通过条件判定与状态切换组织解谜步骤。

实现以“打火机”为核心的交互机制，支持与多类环境对象的动态交互。

负责核心系统与交互架构设计，将场景事件与关卡流程组织为基于事件驱动的模块化结构。

基于 EventBus 解耦交互逻辑、UI、剧情触发与 AI 行为。

使用 Coroutine 管理关卡流程，支持异步加载与状态切换。

基于 NavMesh 实现敌人巡逻与路径约束，并结合 Object Pooling 降低运行时 GC 与实例化开销。

在团队协作中参与问题定位与修复，提升整体流程稳定性。
